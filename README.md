# Systolic Array Simulator

A C++ learning project for modeling a configurable TPU-like systolic array running tiled GEMM workloads.

The project goal is to build a simulator that can report correctness, cycle count, processing-element utilization, and memory traffic for matrix multiplication workloads mapped onto a 2D PE array.

See [systolic_array_project_guidelines.md](systolic_array_project_guidelines.md) for the current project specification and learning roadmap.
